Como conectar no Claude

Em menos de 2 minutos você está gerenciando seu ERP via IA

1️⃣

Cadastre-se

Crie sua conta em aurabr.app/cadastro. Trial de 14 dias grátis, sem cartão de crédito.

2️⃣

Gere sua API Key

No painel ERP, acesse Integrações → MCP Keys e gere sua chave de acesso.

3️⃣

Conecte no Claude

Em Claude.ai → Configurações → Conectores, adicione o MCP Manager com sua URL e API Key.
